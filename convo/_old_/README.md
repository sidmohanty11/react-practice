# convo

## Tech Stack Used

- Vue (frontend)
- Go (Backend)
- MongoDB (Database)
- Pusher (for realtime chatting)

## To work with frontend

`git clone <url>`

`cd convo`

`cd frontend`

`yarn`

`yarn serve`

The server should start up at PORT:8000

## To work with backend

Make sure you have golang and mongodb installed. Run `mongod` in the background.

`cd convo`

`go run main.go`
